Welcome to the Linux Reference Guide

Here are several Linux commands and their definitions.

  • $ uname -a: shows what version of UNIX the computer is running
  • $ passwd: changes the password for the account being used
  • $ man ___: accesses the built-in manual for commands and file formats
  • $ top: displays the running processes
  • $ ls: shows the list directory contents
  • $ cd: changes the directory
  • $ pwd: prints the working directory (where you are right then)
  • $ cat: displays all of the file code
  • $ mkdir: makes a new directory
